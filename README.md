
# Simple Java Docker

A simple java app that runs on docker


## Demo

Clone This Repository

```
git clone https://github.com/HarshalDighe/Simple-Java-Docker.git
```

go to projects

```
cd Simple-Java-Docker
```
Build Docker 

```
 sudo docker build -t mydocker .
```
Run Docker

```
sudo docker run mydocker
```
